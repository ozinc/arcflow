# Skill Operations

Skills derive relationships from world model facts. Register a rule once — the engine materializes edges with confidence scores and provenance, re-evaluable as observations improve.

## Operations

| Operation | What it does |
|---|---|
| [CREATE SKILL](/skills/create-skill) | Register a new skill |
| [PROCESS NODE](/skills/process-node) | Execute skills against nodes |
| [REPROCESS EDGES](/skills/reprocess-edges) | Re-evaluate low-confidence edges |
