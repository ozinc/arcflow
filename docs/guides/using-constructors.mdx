# Using Constructors

Build a self-building graph: register a proximity rule, process nodes, and watch relationships materialize with confidence scores.

## Step 1: Create a constructor

```cypher
CREATE SKILL ProximityDetector
  FROM PROMPT 'Determine if two Person nodes are in the same city'
  ALLOWED ON [Person]
  TIER SYMBOLIC
```

## Step 2: Process nodes

```cypher
PROCESS NODE (n:Person)
```

The engine evaluates `ProximityDetector` against all Person nodes. For each pair
in the same city, it materializes a `NEAR` relationship with confidence and provenance.

## Step 3: Query dynamic edges

```cypher
MATCH (a)-[r]->(b) RETURN a.name, type(r), b.name, r.confidence
```

Dynamic edges carry:
- `confidence` — how certain the constructor is (0.0–1.0)
- `provenance` — which constructor, which version, which model

## Step 4: Re-evaluate low-confidence edges

```cypher
REPROCESS EDGES WHERE confidence < 0.6
```

Deletes weak edges and re-runs their constructors for fresh evaluation.
