# EXPLAIN

See the query plan without executing. Shows the compiled operator pipeline: scan, filter, project, sort steps.

## Syntax

```sql
EXPLAIN <any query>
```

## Example

```sql
EXPLAIN MATCH (n:Person) WHERE n.age > 25 RETURN n.name ORDER BY n.age
```

Returns the compiled operator pipeline: scan, filter, project, sort steps.
