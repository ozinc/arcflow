# PROFILE

Execute with per-operator microsecond timing. Returns results plus performance breakdown for each stage.

## Syntax

```cypher
PROFILE <any query>
```

## Example

```cypher
PROFILE MATCH (n:Person) WHERE n.age > 25 RETURN n.name ORDER BY n.age
```

Returns the query results plus microsecond timing for scan, filter, project,
and sort stages.
