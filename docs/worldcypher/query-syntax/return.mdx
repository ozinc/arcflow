# RETURN

```cypher
MATCH (n:Person)
RETURN n.name AS person, n.age AS years
ORDER BY n.age DESC
LIMIT 10
```

Project query results with aliases, ordering, pagination, and deduplication.

## Syntax

```cypher
RETURN expression [AS alias], ...
  [ORDER BY expression [ASC|DESC]]
  [SKIP n]
  [LIMIT n]
```

## Examples

### Basic projection

```cypher
MATCH (n:Person) RETURN n.name, n.age
```

### Ordering

```cypher
MATCH (n:Person) RETURN n.name ORDER BY n.age DESC
```

### Pagination

```cypher
MATCH (n:Person) RETURN n.name ORDER BY n.age SKIP 5 LIMIT 10
```

### Distinct

```cypher
MATCH (n:Person) RETURN DISTINCT n._labels
```
