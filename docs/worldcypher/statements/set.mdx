# SET

```cypher
MATCH (n) WHERE n.name = 'Alice' SET n.age = 31
```

Update properties on matched nodes or relationships. Automatically updates the `updated_at` timestamp.

## Syntax

```cypher
MATCH (variable) WHERE predicate SET variable.key = value
```
