# Indexes

CREATE INDEX for O(1) property lookups. Without an index, MATCH scans every node.

```cypher
CREATE INDEX ON :Person(name)
```

## Drop an index

```cypher
DROP INDEX ON :Person(name)
```

## List indexes

```cypher
CALL db.indexes()
```

Or in the REPL:

```
:indexes
```
