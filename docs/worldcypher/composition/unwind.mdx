# UNWIND

```cypher
UNWIND ['Alice', 'Bob', 'Carol'] AS name CREATE (n:Person {name: name})
```

Expand a list into individual rows. Each element becomes its own row for downstream processing.

## Examples

### Expand a literal list

```cypher
UNWIND [1, 2, 3] AS item RETURN item
```

### Process collected results

```cypher
MATCH (n:Person) WITH collect(n.name) AS names UNWIND names AS name RETURN name
```
