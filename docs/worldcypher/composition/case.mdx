# CASE WHEN

```cypher
MATCH (n:Person) RETURN n.name,
  CASE WHEN n.age < 18 THEN 'minor'
       WHEN n.age < 65 THEN 'adult'
       ELSE 'senior' END AS category
```

Conditional expressions for computed values in RETURN clauses.

## Syntax

```cypher
CASE WHEN condition THEN result [WHEN condition THEN result]... [ELSE default] END
```

## Examples

### Null handling

```cypher
MATCH (n:Person) RETURN n.name,
  CASE WHEN n.email IS NOT NULL THEN 'has email' ELSE 'no email' END AS status
```
