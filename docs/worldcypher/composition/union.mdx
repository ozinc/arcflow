# UNION

Combine results from two queries into a single result set.

## Syntax

```cypher
query1 UNION query2
```

## Example

```cypher
MATCH (n:Person) RETURN n.name AS name
UNION
MATCH (n:Organization) RETURN n.name AS name
```

Returns all person and organization names in a single column.
