# Get Started

ArcFlow is an embedded graph database. It runs in the browser, Node.js, Python, Rust, and Docker. No server needed.

## Try instantly — zero install

Open **[ArcFlow Engine](https://arcflow.dev/engine)** in your browser. Paste any WorldCypher query. Persistent storage. No install, no signup.

```cypher
CREATE (alice:Person {name: 'Alice', age: 30})
CREATE (bob:Person {name: 'Bob', age: 35})
CREATE (a:Person {name: 'Alice'})-[:KNOWS]->(b:Person {name: 'Bob'})
MATCH (n:Person) RETURN n.name, n.age
CALL algo.pageRank()
```

## Install

```bash
curl -fsSL https://github.com/ozinc/arcflow/releases/latest/download/install.sh | sh
```

Then:

```bash
arcflow --playground
```
