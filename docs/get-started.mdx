# Get Started

ArcFlow is the **World Model database** — built for systems that need a persistent, spatial-temporal, confidence-scored representation of the world.

Space and time are first-class dimensions, not extensions. Every mutation is versioned. Every fact carries a confidence score and an observation class. Relationships are stored objects, not derived at query time. The entire history of the model is queryable with a single syntax — the ISO GQL standard, Cypher-compatible.

As AI systems move from single-agent to multi-agent, they need infrastructure that isn't a retrieval index — they need a world model. Something that knows what connects, where it is, how confident we are, and what it looked like before. ArcFlow is that layer.

It runs in-process. No server. No round-trip. Browser, Node.js, Python, Rust, or Docker — same engine, same queries, same guarantees.

## Try instantly — zero install

Open **[ArcFlow Engine](https://oz.com/engine)** in your browser. Paste any GQL query — persistent storage, no install, no signup.

```cypher
CREATE (e:Entity {id: 'unit-01', x: 12.4, y: 8.7, _observation_class: 'observed', _confidence: 0.94})
CREATE (e:Entity {id: 'contact-x', x: 80.0, y: 90.0, _observation_class: 'predicted', _confidence: 0.38})
CALL algo.nearestNodes(point({x: 0, y: 0}), 'Entity', 5) YIELD node, distance RETURN node.id, distance
MATCH (e:Entity) WHERE e._confidence > 0.8 RETURN e.id, e.x, e.y
MATCH (e:Entity) AS OF seq 1 RETURN e.id, e._confidence
```

## Install

```bash
curl -fsSL https://oz.com/install | sh
```

Then:

```bash
arcflow --playground
```

## All paths

| Path | Time | Friction |
|---|---|---|
| [Browser](https://oz.com/engine) | 0s | Zero — click and go |
| Binary | 60s | `curl ... \| sh && arcflow` |
| npm | 2 min | `npm install arcflow` |
| Docker | 30s | `docker run ghcr.io/ozinc/arcflow:latest` |
| CLI (coding agents) | instant | `arcflow query '...'` |
| MCP (cloud chat UIs) | 1 min | `npx arcflow-mcp` |
