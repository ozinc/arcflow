# Get Started

ArcFlow is a spatial-temporal graph database. Install it and run your first query in under 5 minutes.

## Install

```bash
curl -fsSL https://github.com/ozinc/arcflow/releases/latest/download/install.sh | sh
```

Then:

```bash
arcflow --playground
```

## All install methods

| Path | Time | Command |
|---|---|---|
| Binary | 60s | `curl -fsSL https://github.com/ozinc/arcflow/releases/latest/download/install.sh \| sh` |
| npm | 2 min | `npm install arcflow` |
| Docker | 30s | `docker run -p 7687:7687 ghcr.io/ozinc/arcflow:latest` |
