# Get Started

ArcFlow is the **operational world model layer** — the database that stores, queries, and remembers actual world state for systems that cannot afford to lose spatial precision, temporal fidelity, or epistemic provenance.

The AI infrastructure stack now has three tiers. Foundation models handle language and perception. Neural world models (Cosmos, FSD, Genie) generate action-predicated predictions of how the world evolves. The middle tier — where actual world state persists — has been the missing piece. That is ArcFlow.

Space and time are first-class dimensions, not extensions. Every mutation is versioned. Every fact carries a confidence score and an observation class (`observed`, `inferred`, `predicted`). Relationships are stored objects, not derived at query time. The full history is queryable with a single syntax — the ISO GQL standard, Cypher-compatible.

Neural world models predict. ArcFlow remembers. Every agent in a multi-agent system, every sensor in a fleet, every frame of a camera network writes into and reads from the same queryable world state.

It runs in-process. No server. No round-trip. Browser, Node.js, Python, Rust, or Docker — same engine, same queries, same guarantees.

## Try instantly — zero install

Open **[ArcFlow Engine](https://oz.com/engine)** in your browser. Paste any GQL query — persistent storage, no install, no signup.

```cypher
CREATE (e:Entity {id: 'unit-01', x: 12.4, y: 8.7, _observation_class: 'observed', _confidence: 0.94})
CREATE (e:Entity {id: 'contact-x', x: 80.0, y: 90.0, _observation_class: 'predicted', _confidence: 0.38})
CALL algo.nearestNodes(point({x: 0, y: 0}), 'Entity', 5) YIELD node, distance RETURN node.id, distance
MATCH (e:Entity) WHERE e._confidence > 0.8 RETURN e.id, e.x, e.y
MATCH (e:Entity) AS OF seq 1 RETURN e.id, e._confidence
```

## Install

```bash
curl -fsSL https://oz.com/install | sh
```

Then:

```bash
arcflow --playground
```

## All paths

| Path | Time | Friction |
|---|---|---|

| Docker | 30s | `docker run ghcr.io/ozinc/arcflow:latest` |
