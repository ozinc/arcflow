# Get Started

ArcFlow is the **World Model database** — built for systems that need a persistent, spatial-temporal, confidence-scored representation of the world.

Space and time are first-class dimensions, not extensions. Every mutation is versioned. Every fact carries a confidence score and an observation class. Relationships are stored objects, not derived at query time. The entire history of the model is queryable with a single syntax — the ISO GQL standard, Cypher-compatible.

It runs in-process. No server. No round-trip. Browser, Node.js, Python, Rust, or Docker — same engine, same queries, same guarantees.

## Try instantly — zero install

Open **[ArcFlow Engine](https://arcflow.dev/engine)** in your browser. Paste any Cypher or GQL query — both work. Persistent storage. No install, no signup.

```cypher
CREATE (alice:Person {name: 'Alice', age: 30})
CREATE (bob:Person {name: 'Bob', age: 35})
CREATE (a:Person {name: 'Alice'})-[:KNOWS]->(b:Person {name: 'Bob'})
MATCH (n:Person) RETURN n.name, n.age
CALL algo.pageRank()
```

## Install

```bash
curl -fsSL https://arcflow.dev/install | sh
```

Then:

```bash
arcflow --playground
```

| npm | 2 min | `npm install arcflow` |
| Docker | 30s | `docker run ghcr.io/ozinc/arcflow:latest` |
