# Constructor Operations

Constructors turn static graphs into self-building systems. Register rules, process nodes, and the engine materializes relationships with confidence scores and provenance -- automatically.

## Operations

| Operation | What it does |
|---|---|
| [CREATE CONSTRUCTOR](/constructors/create-constructor) | Register a new constructor |
| [PROCESS NODE](/constructors/process-node) | Execute constructors against nodes |
| [REPROCESS EDGES](/constructors/reprocess-edges) | Re-evaluate low-confidence edges |
