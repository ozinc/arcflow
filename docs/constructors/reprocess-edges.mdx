# REPROCESS EDGES

Re-evaluate low-confidence edges when constructors improve. Deletes edges below a threshold and re-runs their constructors for fresh scores.

## Syntax

```sql
REPROCESS EDGES WHERE confidence < <threshold>
```

## Example

```sql
REPROCESS EDGES WHERE confidence < 0.6
```

This:
1. Finds all dynamic edges with confidence below 0.6
2. Deletes them
3. Re-executes the originating constructors
4. New edges get fresh confidence scores and updated provenance
