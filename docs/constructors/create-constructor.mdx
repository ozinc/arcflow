# CREATE CONSTRUCTOR

```cypher
CREATE SKILL ProfessionalNetwork_v4
  FROM PROMPT 'Given two Person nodes, determine if they have a professional relationship'
  ALLOWED ON [Person, Organization]
  TIER SYMBOLIC
```

Register a relationship constructor. Once created, use [PROCESS NODE](/constructors/process-node) to execute it against matched nodes.

## Syntax

```cypher
CREATE SKILL <name>
  FROM PROMPT '<prompt_template>'
  ALLOWED ON [<Label>, ...]
  TIER <SYMBOLIC|WASM|LLM>
```

## Notes

- `CREATE SKILL` is the GQL statement for registering a constructor
- Constructor is stored in the graph store with a `ConstructorId`
- `TIER SYMBOLIC` is the active execution tier; `WASM` and `LLM` are not yet implemented
