# PROCESS NODE

PROCESS NODE runs all constructors against matched nodes, creating edges with confidence scores and provenance metadata.

## Syntax

```cypher
PROCESS NODE (variable:Label)
```

## Example

```cypher
PROCESS NODE (n:Person)
```

For each matched Person node, the engine:
1. Finds constructors where `allowed_on` includes `Person`
2. Executes each constructor (symbolic tier)
3. Materializes new edges with `Confidence` and `EdgeProvenance`
